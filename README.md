## 🚀 Getting Started

### Requirements
- Node.js 16.x or higher
- npm or yarn
- Git

### Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/aniheaven.git
```

2. Install dependencies:
```bash
cd aniheaven
npm install
```

3. Start the development server:
```bash
npm run dev
```
